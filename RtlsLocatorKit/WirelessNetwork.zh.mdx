# 定位器套件的无线网络

本节详细介绍了RTLS定位器套件无线网络的设置和配置。主要建议在小型演示和技术展示时采用无线安装。对于项目部署或概念验证（PoC），我们强烈建议使用有线解决方案以获得最佳的稳定性和性能。

👇🏻 无线安装**仅建议用于小型演示**和RTLS技术展示。我们建议项目或PoC使用有线解决方案。

## 无线通信流程
RTLS定位器套件在无线使用时，通过锚点之间的UWB通信将位置数据传输到指定的主锚点。然后，该主锚点通过套件中提供的Wi-Fi路由器将数据传回客户端系统（承载RTLS Hub）。

*(占位提示：一个简化的图表，说明数据流：标签 -> 从锚点 (UWB) -> 主锚点 (UWB) -> Wi-Fi路由器 (Wi-Fi) -> 带有RTLS Hub的客户端系统。)*

## 路由器 GL-AR300M 配置

<Callout emoji="ℹ️" type="info">
 **提示：** 您收到的路由器带有制造商的默认配置。
</Callout>

请按照以下步骤配置路由器：
1. 通过micro USB接口将路由器连接到电源。您可以使用任何标准的USB充电器作为电源。使用以太网电缆将您的计算机连接到路由器的 **LAN** 端口。请确保您计算机的以太网接口设置为 **DHCP客户端**。
2. 打开您的网页浏览器并输入以下IP地址：**192.168.8.1**
3. 选择您的首选语言并继续。
4. 设置新的路由器管理员密码：**locusrtls** 并点击提交。
5. 点击 **WIRELESS (无线)**，然后 **Modify (修改)** Wi-Fi设置。
6. 将 **Wi-Fi Name (SSID)** 和 **Wi-Fi Key (密码)** 更改为计划供锚点无线连接使用的值：
    - **Wi-Fi Name (Wi-Fi名称):** Locus RTLS
    - **Wi-Fi Key (Wi-Fi密码):** rtlswifi

*(占位提示：截图说明路由器配置步骤：例如，显示SSID和密钥字段的无线设置页面，LAN IP更改页面。)*

7. 转到 **MORE SETTINGS (更多设置)** 部分，选择 **LAN IP (局域网IP)** 设置。
8. 将默认路由器IP地址更改为：**192.168.88.1**
9. 等待路由器重新启动。如果它没有自动重定向您，请在浏览器中输入新的IP地址：**192.168.88.1**
10. 路由器设置现已完成。新的路由器管理员凭据是：
    - **Username (用户名)**: root
    - **Router Admin Password (路由器管理员密码)**: locusrtls (注意：这是您在步骤4中设置的密码。Wi-Fi密钥 `rtlswifi` 用于设备连接到Wi-Fi网络。)
11. 您现在可以开始部署锚点了。
    
<Callout emoji="ℹ️" type="info">
 **提示：** 您可以将路由器配置保存到文件作为备份。
</Callout>

路由器配置完成且锚点通电后，它们应自动连接到指定的Wi-Fi网络。有关验证其连接和状态的进一步步骤，请参阅锚点文档。
