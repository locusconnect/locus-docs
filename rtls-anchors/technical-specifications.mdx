This section provides detailed technical specifications for Locus Anchors. Specifications may vary between different anchor models and revisions.

## Hardware Specifications

### Physical Characteristics
| Parameter | Value |
|-----------|--------|
| Dimensions | 120 × 80 × 40 mm (typical) |
| Weight | 150g (typical) |
| Housing Material | Industrial-grade plastic |
| Color | White/Gray (standard) |
| Mounting | Wall/ceiling brackets, VESA 75x75mm |

| Parameter | Value |
|-----------|--------|
| Operating Temperature | -20°C to +65°C |
| Storage Temperature | -30°C to +70°C |

|-----------|--------|
