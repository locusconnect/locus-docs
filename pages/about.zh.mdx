# Introduction to JavaScript

JavaScript is a programming language that is used to create interactive effects within web browsers. It is the third layer of the layer cake of standard web technologies, two of which (HTML and CSS) we covered in the first part of this tutorial.

## What is Node.js?

Node.js is a JavaScript runtime built on Chrome's V8 JavaScript engine. Node.js uses an event-driven, non-blocking I/O model that makes it lightweight and efficient. Node.js' package ecosystem, npm, is the largest ecosystem of open source libraries in the world.
