## Floor Plan Requirements

### File Specifications
- Supported formats: PNG, JPG, SVG
- Maximum file size: 10MB
- Recommended resolution: 300 DPI
- Scale: 1:100 or 1:200

### Plan Details
- Clear wall structures
- Room labels
- Scale indicators
- North arrow
- Major landmarks

## Upload Process

1. **Prepare the Floor Plan**
   - Clean unnecessary details
   - Ensure correct orientation
   - Verify scale accuracy
   - Convert to supported format

2. **Import to RTLS Hub**
   - Navigate to Floor Plan Manager
   - Select "Upload New Plan"
   - Choose file from local system
   - Wait for processing

3. **Configure Settings**
   - Set scale reference
   - Define orientation
   - Mark key points
   - Add metadata

## Calibration

### Scale Setting
1. Identify known distance
2. Mark reference points
3. Enter actual measurement
4. Verify scale accuracy

### Orientation
1. Locate true north
2. Align plan accordingly
3. Set reference angle
4. Verify compass alignment

<Callout type="warning" emoji="⚠️">
  Accurate scale and orientation are crucial for precise positioning. Double-check all measurements before finalizing.
</Callout>
