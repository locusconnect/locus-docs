# LC01 Series Tags

## Overview

The LC01 Series represents our foundational line of Ultra-Wideband RTLS tags, designed for reliable asset and personnel tracking in industrial environments.

## Technical Specifications

### Physical Characteristics
| Parameter | Value | Notes |
|-----------|--------|-------|
| Dimensions | 45 × 35 × 15 mm | Compact form factor |
| Weight | 25g with battery | Standard configuration |
| Housing | IP65 rated plastic | Industrial grade |
| Mounting | Various options | Clips, straps, adhesive |

### Radio Specifications
| Parameter | Value | Notes |
|-----------|--------|-------|
| Technology | UWB IEEE 802.15.4z | High precision |
| Frequency | 3.1-4.8 GHz, 6.0-8.5 GHz | Multi-band |
| Channels | 1, 2, 3, 5, 7 | Configurable |
| Range | Up to 50m | Line of sight |
| Update Rate | Up to 10Hz | Configurable |

### Power Specifications
| Parameter | Value | Notes |
|-----------|--------|-------|
| Battery Type | CR2032 | Replaceable |
| Battery Life | Up to 3 years | Usage dependent |
| Sleep Current | < 1µA | Deep sleep |
| Active Current | 20mA typical | During ranging |

## Maintenance

### Battery Replacement
1. Open battery compartment
2. Remove old battery
3. Insert new CR2032
4. Verify LED indication
5. Test tag operation

### Troubleshooting
| Issue | Possible Cause | Solution |
|-------|---------------|----------|
| No updates | Dead battery | Replace battery |
| Intermittent | Poor mounting | Adjust position |
| LED issues | Configuration | Check settings |
| Range issues | Environment | Verify anchors |
