# Locus 锚点技术规格

本节提供了 Locus 锚点的详细技术规格。请注意，不同锚点型号或修订版本（例如，有线与无线版本，如果适用）之间的规格可能略有不同。

## 硬件规格

| 参数                     | 值                                       |
|--------------------------|-------------------------------------------|
| 尺寸                     | 约 120 × 80 × 40 毫米 (因型号而异)        |
| 重量                     | 约 150克 (因型号而异)                     |
| 工作温度范围             | -20°C 至 +65°C                            |
| 电源 (有线型号)          | PoE (兼容 IEEE 802.3af/at)                |
| 电源 (无线型号)          | Micro-USB (5V DC) (如果适用)              |
| 安装选项                 | 壁挂/吸顶安装支架，兼容VESA (例如 Anchor Apollo 为 75x75mm) |
| 外壳防护等级             | IP65 防尘防水等级                         |

## 无线电规格 (UWB)

| 参数                 | 值                                       |
|----------------------|-------------------------------------------|
| UWB 信道             | 支持多个UWB信道 (例如 1, 2, 3, 5, 7)       |
| RF 覆盖范围          | 典型室内范围可达50米 (视环境而定)           |
| 定位精度             | 通常为 10-30厘米 (CEP95)                  |
| 更新速率             | 可配置，最高可达10Hz (或更高，取决于系统配置和标签密度) |
| 有线接口             | RJ45 以太网, 10/100 Mbps (适用于有线型号或主无线锚点) |

## 软件兼容性

- 兼容 Locus 定位引擎 v4.0 及更新版本。
- 通过 Locus RTLS Hub 支持标准和自定义配置文件。
- 固件可通过 Web 界面或 RTLS Hub 升级。

## 认证

认证可能包括 (取决于地区和型号)：
- CE
- FCC
- IC
- 符合 RoHS 标准

_注意：这些规格如有更改，恕不另行通知，此为持续产品开发和改进的一部分。_
